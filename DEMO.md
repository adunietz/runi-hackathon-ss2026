## 90-second script

1. Introduce the patient and problem.
2. Show the starting screen.
3. Complete the central action.
4. Show the unexpected change / intervention.
5. Show the resolution and useful result.
6. Explain expected impact in one line.

## Test checklist

- [ ] App starts from a clean checkout
- [ ] Every button in the demo path works, in order
- [ ] Hebrew/RTL displays correctly (if Hebrew was chosen — see `CLAUDE.md`
      § Language)
- [ ] No real patient information appears anywhere on screen
- [ ] Layout holds up in a mobile viewport
- [ ] Full run-through takes under 90 seconds
